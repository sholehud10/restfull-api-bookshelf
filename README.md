# RESTful API

install semua package

```
npm install
```

jalankan server localhost:5000

```
npm run serve
```

package yang terinstall

```
npm install nodemon --save-dev
npm install eslint --save-dev
npm install @hapi/hapi
npm install nanoid@3.x.x
```

konfigurasi eslint

```
npx eslint --init
```
